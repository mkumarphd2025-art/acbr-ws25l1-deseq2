# Create Conda Environment with Miniconda 
I am creating a conda environment for DESeq2 data anaylysis
with R and Bioconductor in my windows systems using Windows
Subsystem Linux (WSL)

## Code in WSL terminal
- Start the WSL in Windows
- Type the following to create a conda environment for R4.5 names as R4.5WS
```{CMD}
conda create --name R4.5WS
```
- Activate the R4.5WS conda environment as follows
```{CMD}
conda activate R4.5WS
```
